Buttons

Bootstrap does a good job of making all buttons and button states visually appealing from the beginning, making it easy to fall into the flow of making a website. This page will show the basics and many customizations of bootstrap button classes.

Basic Button

The button is set up as it normally would be in regular HTML with a tag of button. Inside the class tag you add btn and what colour you want the button to appear as.

Below each button is listed with it's corresponding colour, with the exception of btn-light and btn-dark.

Example: class="btn btn-danger"

Outlined Button

Similar to a regular button, the outlined button has the same basic shape but instead of the colour coding showing up as the background. It appears as the text colouring and outline.

Below is the same ordering as before but with the addition of outline inbetween the btn-light classing.

Example: class="btn btn-outline-danger"

Small Buttons

Making more compact buttons.

Example: class="btn btn-sm"

Large Buttons

Making bigger buttons.

Example: class="btn btn-lg"

Full Width Buttons

This button will fill the width of the parent container. A common use for this particular one is the submit button on a form.

Example: class="btn btn-lg btn-block"
